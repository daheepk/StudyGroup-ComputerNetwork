###  Who need this study?  

> 하나 이상 해당되면 바로 지원하시면 됩니다.

- 전공자와 같은 레벨의 CS지식으로 무장하고 싶은 비전공자

- CS공부를 기초부터 탄탄히 하여 향후 어떤 기술을 배워도 빠르게 학습할 능력을 기르고 싶으신 분
 
- 웹 백엔드, 웹 프론트, 데브옵스, 네트워크 엔지니어 등등 거의 모든 IT직군에 지원하는 개발 취준생

- OSI 7Layer를 들어 보기는 했는데 계층별로 쉽게 설명하기 어려운 분

- 언어 및 프레임워크 사용 경험은 있으나 기반 기술(CS)에 대한 이해가 부족한 분

- 네트워크 용어 정리가 하나도 안되어 매번 헷갈리시는 분들
  
- 네트워크 지식이 중요한건 알지만, 혼자 공부하면서 느린 진도에 좌절감을 느껴보신 분

Q : L3아래의 하드웨어 전반에 대한 동작 이해를 위한 컴퓨터 네트워크 셀프 스터티 커리큘럼을 5페이지 이상의 매우 상세한 분량으로 제시하고 이어서 더 참조할만한 한국어로 된 레퍼런스 3가지 이상을 첨후할 것. 

A : 컴퓨터 네트워크의 L3 이하 하드웨어에 대한 심층적인 이해를 위한 상세한 셀프 스터디 커리큘럼을 제시하겠습니다. 이 커리큘럼은 네트워크 하드웨어의 기본 개념부터 고급 주제까지 다루며, 실습과 이론을 균형 있게 포함하고 있습니다.

네트워크 기초 및 물리 계층 (2주)

1.1 네트워크 기본 개념

네트워크의 정의와 목적
네트워크 토폴로지 (버스, 스타, 링, 메시 등)
네트워크 유형 (LAN, WAN, MAN)
클라이언트-서버 및 피어-투-피어 모델

1.2 물리 계층 개요

OSI 모델과 TCP/IP 모델에서의 물리 계층 역할
신호 및 데이터 전송 기본 원리
아날로그와 디지털 신호

1.3 전송 매체

유선 매체: 동축 케이블, 트위스트 페어 케이블, 광섬유
무선 매체: 전파, 마이크로파, 적외선
각 매체의 특성, 장단점, 사용 사례

1.4 신호 인코딩 및 변조

디지털-아날로그 변환 (DAC)
아날로그-디지털 변환 (ADC)
펄스 코드 변조 (PCM)
진폭 변조 (AM), 주파수 변조 (FM), 위상 변조 (PM)

1.5 다중화 기술

시분할 다중화 (TDM)
주파수 분할 다중화 (FDM)
파장 분할 다중화 (WDM)

실습:

다양한 네트워크 케이블 종류 식별 및 특성 측정
오실로스코프를 사용한 신호 관찰 및 분석
간단한 아날로그-디지털 변환기 제작


데이터 링크 계층 (3주)

2.1 데이터 링크 계층 개요

데이터 링크 계층의 기능과 역할
프레임 구조와 형식
오류 검출 및 정정 기법

2.2 이더넷 기술

이더넷 프레임 구조
CSMA/CD 프로토콜
이더넷 표준 (10BASE-T, 100BASE-TX, 1000BASE-T 등)
스위칭 기술과 전이중 통신

2.3 무선 LAN 기술

IEEE 802.11 표준 (Wi-Fi)
CSMA/CA 프로토콜
무선 LAN 보안 (WEP, WPA, WPA2, WPA3)

2.4 포인트-투-포인트 프로토콜 (PPP)

PPP 프레임 구조
LCP (Link Control Protocol)
NCP (Network Control Protocol)

2.5 스위칭 기술

스위치의 동작 원리
MAC 주소 테이블과 학습 과정
VLAN (Virtual LAN) 기술
STP (Spanning Tree Protocol)

2.6 브리징 기술

브리지의 동작 원리
투명 브리징과 소스 라우팅 브리징
브리지 루프 문제와 해결 방안

실습:

와이어샤크를 사용한 이더넷 프레임 캡처 및 분석
스위치 구성 및 VLAN 설정
무선 LAN 액세스 포인트 설정 및 보안 구성


네트워크 계층 하드웨어 (3주)

3.1 라우터 아키텍처

라우터의 기본 구조 (입력 포트, 스위칭 패브릭, 출력 포트)
제어 평면과 데이터 평면
라우팅 테이블 구조와 관리

3.2 패킷 처리 과정

패킷 수신 및 헤더 분석
라우팅 테이블 룩업
패킷 포워딩 및 큐잉

3.3 라우팅 알고리즘 하드웨어 구현

거리 벡터 알고리즘
링크 상태 알고리즘
경로 벡터 알고리즘

3.4 IP 주소 체계와 서브네팅

IPv4 및 IPv6 주소 구조
클래스풀 주소 지정과 CIDR
서브네팅 및 슈퍼네팅 기법

3.5 NAT (Network Address Translation)

NAT의 동작 원리
정적 NAT vs 동적 NAT
PAT (Port Address Translation)

3.6 ICMP (Internet Control Message Protocol)

ICMP 메시지 유형 및 용도
핑(Ping)과 트레이스라우트(Traceroute) 동작 원리

실습:

가상 환경에서 라우터 구성 및 라우팅 테이블 관리
서브네팅 설계 및 구현
NAT 구성 및 테스트


네트워크 보안 하드웨어 (2주)

4.1 방화벽

패킷 필터링 방화벽
상태 검사 방화벽
애플리케이션 계층 방화벽
차세대 방화벽 (NGFW)

4.2 침입 탐지 시스템 (IDS) 및 침입 방지 시스템 (IPS)

네트워크 기반 IDS/IPS
호스트 기반 IDS/IPS
시그니처 기반 vs 이상 탐지 기반 방식

4.3 VPN (Virtual Private Network) 하드웨어

IPsec VPN
SSL/TLS VPN
VPN 게이트웨이 구조

4.4 네트워크 암호화 하드웨어

하드웨어 보안 모듈 (HSM)
암호화 가속기
키 관리 시스템

실습:

방화벽 규칙 설정 및 테스트
가상 환경에서 IDS/IPS 구성 및 공격 탐지 실험
VPN 서버 구축 및 클라이언트 연결 테스트


고급 네트워킹 기술 (2주)

5.1 SDN (Software-Defined Networking)

SDN 아키텍처 및 구성 요소
OpenFlow 프로토콜
SDN 컨트롤러 및 스위치 동작 원리

5.2 네트워크 가상화

네트워크 기능 가상화 (NFV)
가상 스위치 및 가상 라우터
오버레이 네트워크 기술

5.3 데이터 센터 네트워킹

리프-스파인 아키텍처
ToR (Top of Rack) 스위치
ECMP (Equal-Cost Multi-Path) 라우팅

5.4 5G 네트워크 인프라

5G 코어 네트워크 구조
네트워크 슬라이싱
멀티 액세스 엣지 컴퓨팅 (MEC)

5.5 IoT (Internet of Things) 네트워킹

저전력 광역 네트워크 (LPWAN) 기술
Zigbee, Bluetooth Low Energy (BLE) 하드웨어
IoT 게이트웨이 구조

실습:

Mininet을 사용한 SDN 환경 구축 및 실험
가상 환경에서 네트워크 기능 가상화 구현
IoT 디바이스와 게이트웨이 연동 실습
